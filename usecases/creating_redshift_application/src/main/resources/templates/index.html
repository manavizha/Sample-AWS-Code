Welcome User to the Amazon Redshift Job Posting example app

Now is:

The Amazon Redshift Job Posting Example application uses multiple AWS Services and the Java V2 API. Perform these steps:

  1. Enter work items into the system by choosing the Add Posts menu item. Fill in the form and then choose Create Item.
  2. The sample application stores the data by using the Amazon Redshift Java API V2.
  3. You can view the items by choosing the Get Posts menu item. Next, select a language.
  4. You can view the items by chooing either the Five Posts, Ten Posts, or All Posts button.
  5. The items appear in the page from newest to oldest.
